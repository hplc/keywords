程序使用说明：

1、本程序是UTF-8编码编写的；
2、上传后必须更改config/config.php里面的数据库信息；
3、有两份sql数据提供导入：
   1、seokeywords.sql 包含有数据的；帐号和密码都是：admin
   2、seokeywords_null.sql 不包含数据的，导入后必须在users表填入帐号和密码
      密码是sha1加密的，所以必须要输入对应的sha1信息
      （百度“sha1加密”就有一些APP应用可以转换，是40位小写字母的）

2012.10.19首发说明
本程序是个人为了方面管理关键词每天排名情况而写的。
1、添加关键词，并且自助加入每个关键词在国内各搜索引擎排名情况。
2、添加网址，自助加入每天收录和反链在各大搜索引擎的情况。
3、集全国内各大搜索引擎： 百度、360搜索、谷歌、搜狗、搜搜、有道、必应、一搜、中搜、盘古搜索、即刻等国内搜索引擎。
4、可自助加入每天的工作日志。
5、后台可更改和删除关键词网页和数据。
6、可查询关键词并高亮显示。

使用前必须导入mysql数据，并且更改好mysql帐号和密码：config/config.php 文件
